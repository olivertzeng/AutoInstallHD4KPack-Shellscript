# AutoInstallHD4KPack-Shellscript
This simply installs 4K or HD packs onto your computer.
You could choose:
Super Mario 3D Land 
Super Mario Sunshine 
Luigi's Mansion
HD and 4K packs by Henriko (https://github.com/HenrikoMagnifico)
(Only Works on Linux Retroarch)
## Installation
`chmod 777 run.sh`
`./run.sh`
